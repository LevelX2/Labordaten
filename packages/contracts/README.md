# Contracts

Dieser Ordner enthält gemeinsame Verträge zwischen Frontend, Backend und späteren externen Importhelfern.

## V1-Ziel

- versionierbares Importschema
- maschinenlesbare Grundlage für Codex- oder KI-nahe Workflows
- kein direkter Datenbankzugriff externer Helfer

## Aktueller Inhalt

- `import-v1.schema.json`: erstes JSON-Schema für strukturierte Befundimporte
