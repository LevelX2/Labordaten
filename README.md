# Labordaten

Dieses Repository ist die technische und fachliche Arbeitsbasis für eine zukünftige Anwendung zur Pflege von Labordaten.

## Aktueller Stand

Die Umgebung ist bewusst zunächst neutral aufgesetzt:

- Git-Repository mit sauberem Grundgerüst
- projektbezogene Wissensbasis unter `ai-project-memory/`
- `AGENTS.md` mit wiki-first Arbeitsweise

## Nächster Schritt

Im nächsten Schritt werden die fachlichen Konzepte, Datenlogiken und Anforderungen für die Laboranwendung aufgenommen und in die Wissensbasis überführt. Danach kann die eigentliche technische Umsetzung vorbereitet werden.
