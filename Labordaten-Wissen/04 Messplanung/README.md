# Messplanung

Dieser Bereich sammelt fachliche Überlegungen dazu, wann welche Laborwerte sinnvoll gemessen werden.

Es geht nicht um Software-Tests, sondern um praktische Messentscheidungen: Welche Werte gehören bei einer Fragestellung zusammen? Welche Kontrollen sind im Verlauf sinnvoll? Welche Parameter sollten nicht isoliert, sondern gemeinsam interpretiert werden?
