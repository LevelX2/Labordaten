# Parameterseiten

In diesem Bereich liegen fachliche Seiten zu einzelnen Laborparametern.

Neue Parameter legen automatisch eine eigene Markdown-Seite an und verknüpfen diese mit dem Parameter. Die Seite kann danach fachlich erweitert werden, zum Beispiel mit Definitionen, eigenen Notizen, Quellenhinweisen, sinnvollen Kombinationen und KSG-Einordnung.
