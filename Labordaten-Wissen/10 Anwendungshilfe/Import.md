# Import

Der Import hilft dabei, Laborwerte aus vorbereiteten Daten in die Anwendung zu übernehmen. Das können strukturierte Daten, manuell vorbereitete Dateien oder extern erzeugte KI-Ergebnisse sein.

Vor der endgültigen Übernahme sollen die Daten geprüft werden. Dadurch können neue Parameter erkannt, Aliasvorschläge bewertet, Parametergruppen vorgeschlagen und offensichtliche Zuordnungsfehler korrigiert werden.

## Möglichkeiten

- Du kannst importierte Messwerte vor dem Speichern fachlich prüfen.
- Du kannst Personen, Befunddaten, Parameterzuordnungen und Einheiten kontrollieren.
- Du kannst neue Parameter übernehmen und automatisch eine Laborwissen-Seite anlegen lassen.
- Du kannst Alias- und Parametergruppenvorschläge aus dem Import für spätere Arbeit nutzbar machen.
