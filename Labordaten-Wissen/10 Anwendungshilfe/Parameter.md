# Parameter

Parameter beschreiben die Stammdaten eines Laborwerts, zum Beispiel Anzeigename, interner Schlüssel, Werttyp, führende Normeinheit, KSG-Einordnung und fachliche Beschreibung.

Zu jedem Parameter kann eine Laborwissen-Seite verknüpft werden. Neue Parameter erhalten automatisch eine solche Markdown-Seite.
