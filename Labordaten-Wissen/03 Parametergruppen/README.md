# Parametergruppen

Dieser Bereich beschreibt fachlich sinnvolle Gruppen von Laborparametern.

Parametergruppen sind keine eigenen Messwerte. Sie helfen dabei, zusammengehörige Werte zu finden, gemeinsam zu planen, im Import vorzuschlagen, in Auswertungen zu filtern und in Berichten verständlich zu bündeln.

Beispiele:
- Blutbild
- Eisenstatus
- Schilddrüsenprofil
- Entzündungsprofil
- Nährstoffdiagnostik
