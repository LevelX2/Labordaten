# Rohquellen

Hier werden unveränderte Rohquellen, Snapshots und Referenzen für das Projekt abgelegt.

Empfohlene Unterordner:

- `repo-root/` für relevante Repository-Snapshots
- `fachkonzepte/` für Anforderungen, Prozesse und Domänenlogik
- `externe-quellen/` für Normen, Fachreferenzen und Webquellen
