# Projekt-Wissensbasis

Diese Wissensbasis dient als dauerhaftes Arbeitsgedächtnis für das Repository `Labordaten`.

## Struktur

- `00 Steuerung`: Regeln und verbindliche Pflegevorgaben
- `01 Rohquellen`: unveränderte Rohquellen, Snapshots und Referenzen
- `02 Wissen`: verdichtetes, verlinktes Projektwissen
- `03 Betrieb`: Log, Qualitätsprüfung und laufende Pflege
- `05 Vorlagen`: Vorlagen für neue Wissensseiten

## Nutzung

1. Neue Quellen zuerst unter `01 Rohquellen` erfassen.
2. Danach Wissen in `02 Wissen` aktualisieren oder anlegen.
3. Änderungen und wichtige Erkenntnisse in `03 Betrieb/Log.md` festhalten.
